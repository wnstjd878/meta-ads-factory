# 메타 광고 소재 공장 — 진행 안내

이 폴더에서 클로드 코드를 열면 이 파일을 자동으로 읽는다.
사용자가 "소재 만들자", "메타 광고 만들자", "시작하자" 같은 말을 하면,
아래 순서를 **한 단계씩** 도와라. 한 단계를 끝내면 결과를 눈으로 확인하고 다음으로 넘어간다.
사용자가 명령어를 몰라도 되게, 무엇을 할지 먼저 한국어로 말하고 그 명령을 대신 실행한다.

## 반드시 지킬 안전 규칙 (가장 중요, 절대 어기지 마라)

- **메타(광고 계정)를 직접 호출하지 마라.** 오직 `python factory.py`(또는 `시작.bat`)를 통해서만 광고를 만든다.
  factory 안의 `guards.py`가 안전장치를 건다: 켜진 광고 생성 금지, 하루 예산 상한(BEP×7) 초과 금지, 총액 예산 금지, 삭제 금지.
  이 안전장치를 우회하는 코드를 새로 짜지 마라. `requests` 로 graph.facebook.com 을 직접 부르는 코드도 짜지 마라.
- **광고를 켜지 마라.** 전부 꺼진(PAUSED) 채로 만들고, 켜는 것은 사람이 광고 관리자에서 직접 한다.
- **토큰·열쇠를 화면에 출력하지 마라.** `.env` 내용을 그대로 보여주지 마라.
- **`guards.py`, `meta.py` 를 수정하지 마라.** 안전장치의 핵심이다.
- **사람 검수 단계(그림을 눈으로 보기)를 건너뛰지 마라.** 그림은 확률이라 사람이 봐야 한다.

## 진행 순서 (한 단계씩)

### 1. 준비 확인
- `python --version` 으로 파이썬이 있는지 확인. 없으면 파이썬 설치를 안내해라
  (python.org 최신 버전 릴리스 페이지 → Files → "Windows installer (64-bit)" .exe, 설치 시 "Add python.exe to PATH" 체크. `.msix` 는 피한다).
- `.env` 파일이 있고 그 안에 `META_ADS_TOKEN` 과 `OPENAI_API_KEY` 두 열쇠가 채워져 있는지 확인
  (값은 출력하지 말고, 키 이름이 있고 비어있지 않은지만 본다). 없으면 `.env.example` 을 복사해 `.env` 를 만들고 열쇠를 넣게 안내한다.

### 2. 잘 되는지 확인 (요금 0원)
- `python check_upload.py` — 이미지 올리기가 되는지
- `python demo_guard.py` — 안전장치가 위험한 호출을 막는지

### 3. 소재 만들기
**네가 실행할 때는 대화형 메뉴 대신 플래그를 써라.** 대화형 `python factory.py` 는
"몇 장?" 처럼 사람 입력을 기다리므로 네가 실행하면 멈춘다. 아래 플래그로 실행한다.
(대화형 메뉴는 사람이 직접 칠 때용이다.)
- (선택) 사용자가 참고할 광고가 있으면 `refs` 폴더에 넣게 하고 → `python factory.py --benchmark` (레퍼런스 분석)
- `python factory.py --images N` (N = 사용자가 말한 장수). 그림이 `out/images` 에 생긴다.
- **여기서 멈추고 사용자에게 `out/images` 폴더를 열어 그림을 직접 보라고 한다.** 이상한 그림(글자 깨짐 등)은 그 파일을 지우게 한다.
- 사용자가 확인을 끝내면 `python factory.py --setup --tag 날짜`. 남은 그림으로 캠페인을 만든다(전부 꺼짐).

### 4. 마무리
- 만들어진 캠페인은 전부 꺼져 있다. 사용자에게 "광고 관리자에서 켤 것만 켜세요"라고 안내한다.
- `out/보고서.html` 을 열어 결과를 보여줄 수 있다.
