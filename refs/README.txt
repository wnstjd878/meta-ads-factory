여기에 "잘 된다고 보는 참고 광고" 이미지를 넣으세요 (png / jpg / webp).

  python factory.py --benchmark

를 실행하면, 클로드가 아니라 그림 생성에 쓰는 같은 열쇠로
각 광고를 4축7키(세계관·타겟·문제·해결 + 카피 논리·소구·시선 흐름 등)로 뜯어봅니다.
결과는 out/refs_analysis.json 에 저장됩니다.

그 다음

  python factory.py --images 6

을 실행하면, 참고 광고의 성공 "구조"가 카피 기획에 자동으로 반영됩니다.
문구를 베끼는 게 아니라 구조만 가져옵니다.

참고 광고가 필요 없으면 이 폴더를 비워두세요. 그러면 기존대로 동작합니다.
