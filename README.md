# 메타 소재 공장

클로드 코드가 광고 소재를 기획하고, 그림을 뽑고, 광고 계정에 올려
캠페인까지 짜둡니다. **전부 꺼진 상태로 만듭니다.** 켜는 건 사람이 합니다.

메타 광고 강의 8강 보충 영상의 실습 코드입니다.

```
1단계   python factory.py --images 6     기획 + 그림 → 폴더에 저장하고 멈춤
        ↓ 사람이 폴더를 열어 그림을 본다. 아닌 건 파일을 지운다
2단계   python factory.py --setup        남은 그림만 광고로
        ↓ 광고 관리자에서 켤 것만 켠다
```

---

## 왜 두 단계인가

그림은 확률입니다. 여섯 장 뽑으면 한두 장은 이상하게 나옵니다. 글자가
깨지거나, 손가락이 여섯 개거나, 분위기가 안 맞거나.

그걸 안 보고 광고로 만들면 이상한 광고에 돈이 나갑니다. 그래서 그림을
뽑고 **멈춥니다.** 폴더를 열어 보시고, 아닌 그림은 **파일을 지우세요.**
지운 소재는 광고가 안 됩니다.

**한 방에 돌리는 실행은 일부러 안 만들었습니다.** 붙이지 마세요.

---

## 안전장치

모든 메타 호출은 `meta.call()` 하나만 지나갑니다. 그 안에서 `guards.py` 가
검사합니다.

| 하려는 일 | 결과 |
|---|---|
| 켜진 상태로 캠페인 생성 | 막힘 |
| 하루 예산이 손익분기 × 7 초과 | 막힘 |
| 총액 예산 사용 | 막힘 |
| 기존 광고 켜기 | 막힘 |
| 삭제 | 막힘 |
| 허용 목록에 없는 경로 | 막힘 |
| 이미지·영상 올리기, 소재·캠페인 만들기 | 통과 (일시중지 자동 삽입) |

만들고 나서 **다시 조회해 정말 꺼져 있는지 확인**합니다. 만들었다는
응답만 믿지 않습니다.

직접 뚫어보실 수 있습니다.

```bash
python -c "from guards import guard; guard('POST','act_1/campaigns',{'status':'ACTIVE'},1300)"
# GuardError: status=ACTIVE 로 만들려 한다. 생성은 PAUSED 만 허용한다.
```

## 준비

### 1. 열쇠 두 개

`.env.example` 을 `.env` 로 복사하고 채웁니다. **채팅창에 붙여넣지 마세요.**

| 값 | 어디서 |
|---|---|
| `META_ADS_TOKEN` | business.facebook.com → 설정 → 사용자 → 시스템 사용자 → 새 토큰 생성. 권한 세 개: `ads_management`, `ads_read`, `business_management` |
| `META_AD_ACCOUNT_ID` | 광고 관리자 주소창의 `act=` 뒤 숫자 |
| `BEP_CENTS` | 손익분기 결과당 비용을 **센트**로. $13 이면 `1300` |
| `OPENAI_API_KEY` | 그림 생성. 장당 요금 |

> **앱을 "게시" 해야 합니다.** 토큰을 만들려면 페이스북 앱이 하나 필요한데,
> 만들기만 하면 "개발 모드"입니다. 그 상태로는 광고 소재를 못 만듭니다.
> `developers.facebook.com` → 앱 설정 → 기본 설정에서 개인정보처리방침
> 주소와 카테고리를 채우고, 왼쪽 **게시** 를 누르세요. 심사와는 다릅니다.

### 2. 내 사업 정보

`config.example.json` 을 `config.json` 으로 복사하고 채웁니다.

```json
{
  "product": "무엇을 파는가",
  "bep": "DB 1건당 $13",
  "page_id": "광고에 쓸 페이스북 페이지 번호",
  "link_url": "광고를 누르면 갈 주소",
  "pixel_id": "픽셀 번호",
  "daily_budget_cents": 2000
}
```

`daily_budget_cents` 는 센트입니다. `2000` = $20. 손익분기 × 7 을 넘으면
시작하기 전에 멈춥니다.

### 3. 스킬 파일

`skill/SKILL.md` 를 `~/.claude/skills/meta-ads-ops/SKILL.md` 로 복사하세요.
카피 만드는 규칙이 여기 들어 있습니다. 클로드 코드가 이걸 읽고 씁니다.

같은 파일을 claude.ai 나 Cowork 에도 올리면 `/판정` `/소재생성` `/채점` 을
쓸 수 있습니다.

## 전체를 돌리기 전에 하나씩 확인

처음부터 다 돌리면 어디서 막혔는지 알기 어렵습니다.

```bash
python check_upload.py    # 토큰 + 이미지 올리기 (요금 0원, 30초)
python check_setup.py     # 가드 + 캠페인 만들기 (요금 0원)
python check_image.py     # 그림 한 장 (한 장 요금)
```

`check_upload.py` 가 통과하면 이 프로그램의 핵심이 되는 겁니다. 메타 공식
연결(MCP)로는 이미지를 올릴 수가 없어서, 이 코드가 존재하는 이유가 그
한 줄입니다.

## 실행

```bash
python factory.py --images 3
```

`out/images/` 에 PNG 가 떨어집니다. 광고 계정은 안 건드립니다.
그림을 보고, 아닌 건 지우고, 그다음:

```bash
python factory.py --setup --tag 0709
```

끝나면 `out/보고서.html` 을 더블클릭하세요. 만든 그림과 카피가 한 화면에
뜹니다. 인터넷 없이도 열립니다.

### 영상도 됩니다

`out/videos/01.mp4` 처럼 **카피 번호에 맞춰** 영상을 넣어두면, 그 번호는
이미지 대신 영상으로 광고가 만들어집니다. 문구는 `plan.json` 의 후킹과
서브를 그대로 씁니다.

영상 만들기 자체는 이 프로그램이 안 합니다. 다른 도구로 만들어 넣으세요.
썸네일은 메타가 첫 프레임으로 자동 생성합니다.

## 주 1회 자동으로

**1단계만** 자동으로 겁니다. 2단계는 사람이 칩니다.

```powershell
schtasks /create /tn "메타소재공장" ^
  /tr "python C:\경로\meta-ads-factory\factory.py --images 6" ^
  /sc weekly /d MON /st 03:00
```

컴퓨터가 절전에 들어가지 않게 해두세요.

아침에 폴더를 열어 그림을 보고, 이상한 건 지우고, `--setup` 한 줄.
30초 걸립니다. **이 30초를 아끼려고 하지 마세요.**

## 돈이 나가는 곳

1. **그림 생성**: 장당 요금. `image.py` 의 `MAX_IMAGES = 8` 이 상한입니다.
2. **광고 집행**: 0원. 만들기만 하고 켜지 않습니다.

광고 관리자에서 켜진 캠페인을 발견하면 안전장치가 뚫린 겁니다. 즉시 끄고
알려주세요.

---

## 일부러 안 하는 것

- **광고 켜기.** 사람이 그림을 보고 켭니다.
- **삭제.** 잘못 지우면 복구가 안 됩니다.
- **한 방 실행.** 두 단계 구조 자체가 안전장치입니다.
- **ABO/CBO 승격.** 111 테스트를 통과한 뒤 사람이 판단합니다.
- **성과 판정.** 그건 claude.ai 에서 `/판정` 으로 봅니다.
- **알림 서비스.** 열쇠를 더 늘리지 않습니다. `out/보고서.html` 을 여세요.

## 파일

```
guards.py         안전장치
meta.py           메타를 부르는 유일한 통로
steps/plan.py     ① 소재 기획 (클로드 + 스킬)
steps/image.py    ② 그림 생성
steps/upload.py   ③ 이미지·영상 올리기
steps/setup.py    ④ 111 캠페인 (전부 꺼짐)
steps/report.py   ⑤ 보고서
factory.py        1단계와 2단계
skill/SKILL.md    카피 만드는 규칙

check_upload.py   이미지 올리기만 시험
check_setup.py    캠페인 만들기만 시험
check_image.py    그림 한 장만 시험
```

만질 파일은 `.env` 와 `config.json` 두 개뿐입니다.
