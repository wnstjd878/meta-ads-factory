2. 열쇠 넣기 (.env 파일)
--------------------------------------------------------

# .env.example 을 복사해 .env 를 만들고 메모장으로 연다
copy .env.example .env
notepad .env

# 메모장에 열쇠 두 개를 넣고 저장:
#   META_ADS_TOKEN=메타_시스템사용자_토큰
#   META_AD_ACCOUNT_ID=광고계정_숫자만
#   OPENAI_API_KEY=그림생성_열쇠
#   BEP_CENTS=손익분기_결과당비용 (예: $13 이면 1300 / 2만원이면 20000)

--------------------------------------------------------
4. 소재 만들기 (번호 메뉴)
--------------------------------------------------------

# 이 한 줄만 치면 번호 메뉴가 뜬다. 번호만 고르면 된다.
python factory.py

#   0 = 참고 광고 분석
#       잘 되는 참고 광고를 refs 폴더에 넣어두면 4축7키로 뜯어본다.
#       그 성공 구조가 다음 카피에 반영된다 (문구는 안 베낌). 없으면 안 눌러도 됨.
#   1 = 카피 + 그림 뽑기
#       "몇 장?" 하고 물으면 숫자를 넣는다. 그림이 out/images 폴더에 쌓인다.
#   2 = 광고 만들기 (전부 꺼진 채로)
#       남은 그림으로 캠페인을 만든다. 돈은 안 나간다.

# 순서: 먼저 1번으로 그림을 뽑는다
#       -> out/images 폴더를 열어 눈으로 본다
#       -> 이상한 그림(글자 깨짐 등)은 그 파일을 지운다 (그게 검수)
#       -> 다시 python factory.py 를 치고 2번을 고른다
#       -> 광고 관리자에서 켤 것만 켠다 (여기서 처음 돈이 나간다)


--------------------------------------------------------
5. 매주 자동으로 돌리기 (선택)
--------------------------------------------------------

# 월요일 새벽 3시에 1단계(그림 뽑기)만 자동 실행.
# C:\경로 부분을 자기 폴더 경로로 바꿔서 한 줄로 친다.
# (자동 실행은 옆에서 번호를 눌러줄 사람이 없어서 옛 방식으로 적는다.
#  --images 6 이 "1번을 골라서 여섯 장" 과 같은 뜻이다.)
schtasks /create /tn "메타소재공장" /tr "python C:\경로\meta-ads-factory\factory.py --images 6" /sc weekly /d MON /st 03:00

# 2단계(광고 만들기)는 자동으로 걸지 않는다. 그림은 사람이 봐야 하기 때문.
